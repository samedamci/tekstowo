# tekstowo

Retrieve lyrics or translation of a given song from
[tekstowo.pl](http://www.tekstowo.pl/).

### Example

```shell
$ tekstowo.py "Nirvana - About a girl"
I need an easy friend
I do, with an ear to lend
...
```
